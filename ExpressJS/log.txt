//Versioning

Current Version -> ^4.18.2 

1st Part -> 4
2nd Part -> 18
3rd Part -> 2

3rd Part (Last Part) - Minor Fixes (Optional)

2nd Part (Middle Part) - Bug Fix Security Update (Recommended)

1st Part (First Part) - Major/Breaking Update -> Previous code might not work as the syntax or other things might change.

^ -> Compatible with Version - Version remains in 4 range i.e. 4.18.2 - <5.0.0
~ -> Approximately Equivalent to Version - Only the last Minor Fixes are Updated i.e. 4.18.2 - <4.19.0
There are Various other ways to use Version.

To Download a Specific Version of Express - npm install express@<version>
